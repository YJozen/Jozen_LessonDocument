課題①  
ひとまず「Readme.md」には
```md
# 自分たちの作品名
　作品について

<br>

# チームメンバー
　出席番号_チームメンバー

<br>

# 進捗,予定,メモなど
　〜内容〜
```
といった形式で書いて下さい

Readme.mdはマークダウンというファイル形式になり、「マークダウン」という記法を使用します。
htmlのタグやcssも使用可能です。
慣れたら記述の仕方にも気をつけて下さい。
